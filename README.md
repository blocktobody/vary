## 이메일 마케팅 솔루션 배리를 만들면서 배운 것들

해당 레포는 2019년 8월 1일~2020년 8월 31일까지 이메일 마케팅 솔루션 [배리](https://vary.one)의 개발팀에서 프론트엔드 직무를 담당하며 경험하고 배운 것들을 정리하는 곳입니다. 정리할 내용들은 다음과 같습니다.

- 배리 프론트엔드에서 사용한 기술 스택 관련
  - Typescript
  - React
  - Redux
  - redux-observable
  - rxjs
  - next.js
- 배리 에디터 제작시 경험한 프로젝트 구조와 서드파티 라이브러리 관련
  - presentational과 container
  - MVC와 Flux
  - storybook
  - WYSIWYG 에디터
- 기타 일반적인 주제들
  - 성능 개선
  - 이메일 코딩을 위한 마크업과 스타일
  - 비전공자가 개발을 공부하는 방법
